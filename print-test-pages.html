Print Test Pages Instructions

  1. Print Settings: Set scale to 100% (no "fit to page")
  2. Margins: Set to "None" or "Minimum"
  3. Orientation: Portrait
  4. Paper Size: US Letter (8.5" × 11")
  5. Print Preview: Check that alignment marks are visible at corners
  6. After Printing: Use a ruler to verify measurements

macOS Terminal Commands:

# View printers
lpstat -p -d

# Print this file
